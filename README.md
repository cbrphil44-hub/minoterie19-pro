# Application Recette 19 V12.3 Admin complet

Objectif : tableau administrateur complet sans toucher aux recettes, calculs, familles ni sauvegardes existantes.

Inclus :
- Mode administrateur par appui long 5 secondes sur le logo + code 1702.
- Gestion des accès : liste, accepter, refuser, remettre en attente, suspendre, supprimer, recherche, tri.
- Rapport d'erreurs : date/heure, type, message, bouton vider.
- Gestion de l'application : créer une mise à jour, clients autorisés, demandes en attente, comptes suspendus.
- Sauvegardes : sauvegarde complète, restauration, export données, import données.
- Informations : version, base Firebase, dernière sauvegarde.
- Protection des recettes : récupération automatique de l'ancienne base locale la plus complète.
